# 🎯 Shooting Game

An interactive web-based shooting game built with HTML5 Canvas and Vanilla JavaScript.

## 🎮 How to Play

- **Move your mouse** to aim the gun
- **Click** to shoot at targets
- **Press Space** to restart the game

## 🎯 Game Features

- **Dynamic Targets**: Red targets appear on screen and move around
- **Bonus Targets**: Golden targets are worth 10 points instead of 1
- **Increasing Difficulty**: More targets spawn as your score increases
- **Score Tracking**: Real-time score display
- **Smooth Controls**: Mouse-based aiming with crosshair cursor
- **Visual Effects**: Glowing projectiles and shadows for better gameplay

## 📊 Scoring System

- **Red Target**: 1 point
- **Gold Target**: 10 points (bonus!)
- **Difficulty Escalation**: More targets spawn as you progress

## 🚀 Getting Started

1. Open `index.html` in your web browser
2. Aim using your mouse
3. Click to shoot
4. Rack up points by hitting targets!

## 🛠️ Technologies Used

- **HTML5**: Canvas API for graphics
- **CSS3**: Responsive styling with gradients
- **Vanilla JavaScript**: Pure JS game logic

Enjoy the game! 🎯
